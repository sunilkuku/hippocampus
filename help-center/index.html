Student Register And Login

  1. Click Register
  2. Use Your mobile number as username
  3. Password will be sent to your phone through text message
  4. After Registration user can log in using the mobile number as username and password which was sent to your mobile number.

Password Reset

  1. Goto Myaccount->Password Reset, menu
  2. Enter the mobile number as username and email address to verify the user.
  3. A new password will be sent to your phone through text

Assignment Submission

How to View Assignment Topics 

Register And Login  First To View Assignment Menu

  1. Go to Assignment -> View Assignments 
  2. Select Department-> Semester->Subject Code->Click   

How to Submit Assignments 

 Register And Login  First To View Assignment Menu

  1.  Go to Assignment -> Submit My Assignment.
  2. Enter Title as “Subject Code – Subject Name – University Register Number – Student Name”.
  3. Type Assignment  In The Body.
  4. You can also Upload PDF & DOC Files by Clicking ” Add Media ”  (Available on both desktop and mobile)
  5. Choose Correct Subject Code From The Menu Under The Editor. (If wrong subject code is selected then delete the current assignment and add a new one with correct subject code).
  6. Click Publish Or Click Draft For Later Publishing.
 

How to View & Edit My Assignments

 Register And Login  First To View Assignment Menu 

1. Go to Assignment -> View My Assignments
